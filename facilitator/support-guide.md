# 参加者サポートガイド

## 🎯 個別サポートの基本方針

### サポートの優先順位
1. **安全確保** - 最優先事項
2. **全体進行の維持** - スケジュール管理
3. **個別学習支援** - 参加者の理解促進
4. **技術的問題解決** - 機器・設定の問題対応

### サポートレベルの調整
参加者のスキルレベルに応じてサポート方法を調整：

#### 初心者レベル
- **特徴**: 技術的な知識が少ない、不安が大きい
- **サポート方法**: 
  - 丁寧な説明と確認
  - 一緒に操作を実施
  - 頻繁な進捗確認

#### 中級者レベル
- **特徴**: 基本的な操作は理解、応用に不安
- **サポート方法**:
  - ポイントを絞った説明
  - 自主的な作業を促進
  - 必要時のサポート提供

#### 上級者レベル
- **特徴**: 技術的理解が高い、自立的
- **サポート方法**:
  - 最小限の説明
  - 他の参加者のサポート役を依頼
  - 発展的な内容の提供

## 📋 進捗確認のチェックポイント

### GPS機器セットアップフェーズ

#### 5分経過時点
**確認項目**:
- [ ] 機器の電源が入っている
- [ ] 基本画面が表示されている
- [ ] 操作方法を理解している

**遅れている場合の対応**:
1. 個別に近づいて状況確認
2. 機器の状態をチェック
3. 必要に応じて機器交換を検討

#### 15分経過時点
**確認項目**:
- [ ] 初期設定が完了している
- [ ] ネットワーク接続設定中
- [ ] エラーが発生していない

**遅れている場合の対応**:
1. 設定内容の再確認
2. 他の参加者とのペア作業を提案
3. 技術サポート担当者への相談

#### 25分経過時点
**確認項目**:
- [ ] 全ての設定が完了している
- [ ] 動作確認の準備ができている

**遅れている場合の対応**:
1. 予備機器への交換を実施
2. 設定済み機器の提供を検討
3. 次フェーズでの追いつき計画

### 動作確認フェーズ

#### 確認すべきポイント
- GPS信号の受信状況
- データ送信の成功
- 機器の動作安定性

#### 問題発生時の対応手順
1. **症状の確認**: 何が起きているかを正確に把握
2. **基本対応の実施**: トラブルシューティングガイドに従った対応
3. **エスカレーション判断**: 5分以内に解決しない場合は技術サポートへ

### 街歩きフェーズ

#### 安全管理のチェックポイント
- [ ] 全参加者の位置確認
- [ ] 交通安全の注意喚起
- [ ] 機器の取扱い状況確認
- [ ] 体調不良者の有無確認

#### 技術サポートのポイント
- GPS機器の動作状況確認
- データ記録の継続確認
- バッテリー残量の確認

## 🆘 問題エスカレーションの手順

### レベル1: 司会進行者による対応
**対象**: 基本的な操作問題、軽微な設定問題
**対応時間**: 3-5分以内
**対応方法**: 
- トラブルシューティングガイドの参照
- 基本的な再設定の実施
- 参加者への操作指導

### レベル2: 技術サポート担当者への相談
**対象**: 複雑な技術問題、機器の不具合
**エスカレーション基準**:
- 5分以上解決しない問題
- 同じ問題が複数の参加者で発生
- 機器の故障が疑われる場合

**対応方法**:
1. 問題の詳細を技術担当者に報告
2. 技術担当者による直接対応
3. 必要に応じて機器交換

### レベル3: 全体進行の調整
**対象**: 全体スケジュールに影響する問題
**調整基準**:
- 30%以上の参加者が遅れている
- 安全上の問題が発生
- 外的要因による変更が必要

**対応方法**:
1. 全体への状況説明
2. スケジュールの調整
3. 代替手順の実施

## 💡 効果的なサポートのコツ

### コミュニケーション
- **明確で簡潔な説明**: 専門用語を避け、分かりやすい言葉を使用
- **積極的な声かけ**: 困っている様子を見つけたら積極的にサポート
- **肯定的な態度**: 参加者の努力を認め、励ましの言葉をかける

### 技術サポート
- **段階的な説明**: 一度に多くの情報を与えず、段階的に説明
- **実演による指導**: 口頭説明だけでなく、実際に操作して見せる
- **確認の徹底**: 参加者が理解したかを必ず確認

### 時間管理
- **優先順位の判断**: 全体進行への影響を考慮した対応順序
- **並行作業の活用**: 他の参加者の作業を止めない工夫
- **柔軟な調整**: 状況に応じたスケジュール調整

## 📊 サポート記録の取り方

### 記録すべき項目
- 発生した問題の内容
- 対応にかかった時間
- 解決方法
- 参加者の反応

### 記録の活用方法
- 次回ハンズオンの改善
- よくある問題の事前対策
- サポート手順の改善
- 資料の更新
