# 司会進行者向けガイド

## 📋 概要

このガイドは「ハンズオン！益田の街を歩こう！ GPS機器をセットアップして駅周辺を散策」の司会進行者向けの資料です。

## 🎯 司会進行者の役割

### 主要な責任
- ハンズオン全体の時間管理
- 参加者への適切なガイダンス提供
- 技術的問題発生時のサポート調整
- 安全管理と注意喚起

### 成功の指標
- 全参加者がGPS機器のセットアップを完了
- 安全に街歩きを実施
- 参加者が満足できる学習体験を提供

## 🔧 事前準備

### 機材確認
- [ ] GPS機器の動作確認
- [ ] 予備機器の準備
- [ ] 充電器・ケーブル類の準備
- [ ] 配布資料の印刷

### 会場準備
- [ ] 机・椅子の配置
- [ ] 電源の確保
- [ ] Wi-Fi環境の確認
- [ ] 緊急連絡先の確認

### スタッフ準備
- [ ] 役割分担の確認
- [ ] 技術サポート担当の配置
- [ ] 安全管理担当の配置

## 🎤 進行のポイント

### 開始時
1. 参加者の出席確認
2. ハンズオンの目的と流れの説明
3. 安全に関する注意事項の伝達
4. 質問・サポートの受け方の説明

### 進行中
- 定期的な進捗確認
- 遅れている参加者への個別サポート
- 時間調整の実施
- 安全確認の継続

### 終了時
- 機材の回収確認
- 参加者からのフィードバック収集
- 次回への改善点の記録

## 📞 緊急時対応

### 技術的問題
1. まずトラブルシューティングガイドを確認
2. 技術サポート担当者に相談
3. 必要に応じて予備機器に交換
4. 代替手順での継続を検討

### 安全上の問題
1. 即座に活動を停止
2. 参加者の安全確保
3. 必要に応じて緊急連絡
4. 状況に応じた対応の実施

## 📈 改善のための記録

### 記録すべき項目
- 参加者の完了率
- 発生した問題とその対応
- 時間配分の実績
- 参加者からのフィードバック

### 次回への活用
- 問題点の改善策検討
- 資料の更新
- 進行方法の改善
- 機材・環境の改善
