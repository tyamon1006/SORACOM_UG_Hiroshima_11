# SORACOM ハンズオン：益田の街を歩こう！

GPS機器をセットアップして駅周辺を散策するハンズオンガイド

## 🎯 このハンズオンについて

SORACOM UG 広島 #11 × IWAMI TECH COMMONSイベントで実施する「ハンズオン！益田の街を歩こう！ GPS機器をセットアップして駅周辺を散策」の参加者向けガイドです。

このハンズオンでは、GPS機器のセットアップから実際の街歩きまでを体験し、IoTデバイスを使った位置情報の取得と活用方法を学びます。

## 📋 当日の流れ

1. **GPS機器セットアップ**
2. **動作確認**
3. **益田街歩き**
4. **データ確認・まとめ**

### ⚠️ 街歩き時の注意事項
- **メール通知の設定**: 益田街歩き中にパソコンを開いて確認するのは危険ですので、メールの通知先は手元のスマートフォンで確認可能なメールアドレスに設定してください。
- **安全第一**: 歩行中はスマートフォンの画面を見続けず、周囲の安全を確認しながら散策しましょう。
