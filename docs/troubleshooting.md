# トラブルシューティングガイド

## 🔧 よくある問題と解決方法

### GPS関連の問題

#### GPS信号が受信できない
**症状**: GPS機器が位置情報を取得できない

**解決手順**:
1. 屋外の開けた場所に移動する
2. 機器を水平に保持する
3. 高い建物や障害物から離れる
4. 2-3分待って信号の安定を確認
5. それでも改善しない場合はスタッフに相談

#### 位置情報の精度が悪い
**症状**: 取得される位置が実際の場所と大きく異なる

**解決手順**:
1. GPS機器の向きを調整する
2. 金属製の物体から離す
3. 機器を再起動する
4. 衛星の受信状況を確認する

### 接続・通信の問題

#### データが送信されない
**症状**: GPS機器からデータが送信されない

**解決手順**:
1. ネットワーク接続状況を確認
2. SIMカードの挿入状態を確認
3. 機器の設定を再確認
4. 機器を再起動
5. スタッフに設定の確認を依頼

#### 機器が起動しない
**症状**: GPS機器の電源が入らない

**解決手順**:
1. バッテリー残量を確認
2. 充電ケーブルの接続を確認
3. 電源ボタンの長押し（5秒程度）
4. 別の機器との交換をスタッフに依頼

## 🆘 サポート要請方法

### スタッフへの連絡方法
1. **直接声かけ**: 会場内のスタッフに直接相談
2. **挙手**: 困った時は手を上げてスタッフを呼ぶ
3. **グループ内相談**: 同じテーブルの参加者と情報共有

### 問題報告のフォーマット
スタッフに相談する際は、以下の情報をお伝えください：

- **何をしていた時に問題が発生したか**
- **どのような症状か**
- **エラーメッセージがあれば内容**
- **試した解決方法があれば内容**

### 代替手順
機器の問題が解決しない場合：

1. **予備機器への交換**: スタッフが予備機器を用意
2. **ペア作業**: 他の参加者と一緒に作業継続
3. **デモンストレーション**: スタッフによる実演で学習継続

## 📞 緊急連絡先

- **会場スタッフ**: 会場内で直接対応
- **技術サポート**: 会場内の技術担当者
- **医療関連**: 体調不良時は会場スタッフまで

## 💡 予防のためのヒント

- 機器は丁寧に扱う
- バッテリー残量をこまめに確認
- 不明な点は早めにスタッフに相談
- 無理な操作は避ける
